# PUBGAssistance C++ 使用说明

主程序窗口分为四个选项卡：**地图点位**、**启动助手**、**校准区域**、**按键设置**。

---

## 1. 使用前准备

### 1.1 推荐环境

- 系统环境 Windows 11。
- 游戏语言选择中文（武器名称和装备栏识别依赖中文）。
- 游戏显示模式：全屏或无边框全屏（保证截图区域位置稳定）。
- 更换分辨率、缩放比例、游戏 UI 比例或显示器后，需重新校准区域。

### 1.2 游戏内必做设置

以下设置不完成将导致功能异常：

1. **设置 → 游戏玩法 → 武器装备栏 → 禁用**：禁用后血条上方出现武器图标，供武器识别使用。
2. **设置 → 游戏玩法 → 小地图动态缩放 → 禁用**：防止小地图测距时比例尺变化。
3. **游戏内所有灵敏度调回默认值**：压枪、VSS、十字弩、火箭筒等准星移动类功能都按默认灵敏度校准；如果游戏内灵敏度不是默认值，实际效果会偏差。
4. **色盲模式**：若游戏开启了色盲模式，需在助手「地图点位」选项卡中同步设置。
5. **打开装备栏按键保持一致**：如果你在游戏里把打开装备栏从 `Tab` 改成了 `Enter` 或其他按键，必须在助手「按键设置」中同步修改「打开装备栏」。
6. **开火按键保持一致**：游戏内开火键和助手「按键设置」中的「开火按键」必须一致，否则压枪、连点、投掷自动释放等操作会异常。
7. **F1~F8 按键尽量不要触发系统指令**：如果这些键被系统或其他软件占用，建议用 PowerToys 等工具映射为不冲突的虚拟按键，并在助手里重新录制对应快捷键。

### 1.3 程序文件要求

主程序需要与以下文件在同一目录下：

- `config/config.json`：保存校准区域、快捷键、色盲模式、标点颜色等配置。
- `assets/templates/`：保存武器、配件、标点等识别模板，不建议随意删除或改名。
- `config/recoil_settings.json`：压枪参数配置。
- `config/special_assistants.json`：特殊武器参数配置。
- `config/map_data.json`：地图点位数据。

## 2. 界面概览

程序运行后，屏幕左上角出现半透明主窗口，左下角出现状态 HUD。开启测距后会叠加测距显示层。

### 2.1 主窗口

- 最上端有四个选项卡，右侧红色圆点可关闭程序。
- 按 `Home` 显示/隐藏主窗口。隐藏后状态 HUD 和已开启的辅助功能继续工作。
- 按 `←` / `→` 可快速切换选项卡。

### 2.2 状态 HUD（左下角）

恒定显示以下信息：

- **第一行**：武器识别、测距显示、辅助压枪的开关状态（绿色开启、红色关闭），以及装备栏是否打开。
- **第二/三行**：装备栏中识别到的两把主武器及配件。
- **第四行**：当前手持武器和姿势；识别到霰弹枪并启用压枪时，还会显示当前探头方向。
- 开启测距后，状态栏上方还会显示当前使用的标点颜色（供手雷瞬爆、C4 冲锋等自动操作使用）。

### 2.3 测距显示层

开启测距后，小地图右上端显示两排标点：上排为大地图测距数据，下排为小地图测距数据。

## 4. 首次使用：区域校准

第一次使用时，必须先完成区域校准，再开启辅助功能。

### 4.1 校准步骤

1. 启动游戏并进入训练场或对局。启动助手，进入「校准区域」选项卡。
2. 点击 **显示所有校准区域框**，查看各区域位置是否匹配。
3. 若不匹配，逐个点击下方按钮进行框选校准：左键从一个角拖到对角完成校准；误触可用右键取消。
4. 每个区域框在满足校准原则的前提下，**框越小计算越快**。
5. 完成后重启助手，使所有配置稳定生效。

### 4.2 各区域校准原则

| 区域 | 校准方法 |
| --- | --- |
| **小地图 / 大地图** | 按 `N` 打开小地图、`M` 打开大地图。从地图左上角边界拖到右下角边界。程序会强制拖成正方形，找准左上角和一个边即可。注意不要将小地图外轮廓框入区域内。 |
| **1km 比例尺** | 打开大地图，缩放到迫击炮测距习惯的比例，从地图上 1km 正方形的一个边起点拖到终点，标定 1km 的真实屏幕距离。此后迫击炮大地图测距必须使用同一缩放比例。 |
| **武器 1/2 编号** | 按你的「打开装备栏」按键打开装备栏，两个主武器名称左侧分别有方框「1」和「2」，框住它（大小比外侧白框大一像素）。可再次按装备栏键查看状态栏是否显示"武器识别中…"来验证。 |
| **武器 1/2 名称** | 比武器名称文字略大一些。框好后需在「调试缩放比例」中校准缩放比例才能正常识别。 |
| **武器 1/2 配件**（倍镜/枪口/握把/枪托） | 分别选择对应区域框，大小比外侧白框大一像素。打开装备栏检查是否识别到配件（前提是武器名称已能识别）。 |
| **武器图标** | 关闭武器装备栏后，血条上方的武器图标区域，比图标大一圈即可。框好后需在「调试缩放比例」中校准缩放比例。 |
| **姿势区域** | 第一人称下血条左侧的姿势图标（站立/蹲/趴），框选该区域。框好后需在「调试缩放比例」中校准缩放比例。 |
| **垂直测高** | 屏幕中间的一竖长条，用于迫击炮测距高度补偿。宽度大概即可，上端不要选到顶部罗盘。 |
| **四倍/六倍/八倍镜内边** | 打开对应倍镜，找到倍镜内框在上方的边界并框选。确保呼吸晃动时内框不超出区域，且不要框到倍镜外框边缘（区域内最上端保持是黑色）。 |

### 4.3 缩放比例校准

如果模板识别不稳定，进入「校准区域」→ **打开缩放校准窗口**：

1. 选择待校准区域（四个区域均需校准：武器名称、武器图标、姿势、武器编号）。
2. 调出该区域内的图标（武器名称需打开装备栏并捡起两把主武器；武器图标需退出装备栏并手持该武器；姿势区域切换第一人称即可）。
3. 选择与当前匹配的模板，右侧可预览。模板与缩放后截图的尺寸越接近，匹配结果越好。
4. 点击「自动寻找 XY 缩放比例」自动搜索最佳组合，或手动拖动滑块/直接输入数值。
5. 点击保存。依次校准完四个缩放比例后，可调出武器和姿势进行测试验证。

| 选项 | 说明 |
| --- | --- |
| **选择地图** | 支持：艾伦格、米拉玛、泰戈、荣都、帝斯顿、维寒迪。进入新地图前必须切换，否则点位不匹配。 |

**显示地图点位**：当前未识别到手持武器时，按住 `鼠标左键 + 中键` 显示点位；按 `鼠标右键` 关闭。若识别到手持武器，点位助手会自动关闭以避免影响战斗。

### 5.2 启动助手

| 按钮 | 快捷键 | 说明 |
| --- | --- | --- |
| **武器检测** | `F1` | 开启后启用装备栏识别、手持武器识别、姿势识别。关闭后当前武器被清空，压枪和特殊武器助手失去基础信息。**日常建议保持开启**。 |
| **瞄准辅助** | `F2` | 开启后显示小地图测距、垂直测高、大地图测距、迫击炮/特殊武器 HUD。关闭后所有特殊武器显示层关闭。**特殊武器助手不显示时优先检查此项**。 |

| **重新加载压枪配置** | — | 手动修改过压枪参数后点击即可生效，无需重启。 |

| **调试特殊武器** | — | 打开特殊武器参数编辑器，详见 [特殊武器校准](#7-特殊武器校准)|

#### 特殊武器助手

特殊武器助手（迫击炮、火箭筒、投掷物、VSS、十字弩、C4）需要同时满足：**瞄准辅助已开启**、**对应助手按钮处于开启状态**，且 **当前武器识别正确**。

- 启动助手中的六个特殊武器按钮默认都是开启状态，表示允许识别到该武器时自动启动对应助手。
- 点击某个按钮会关闭该助手；即使后续识别到该武器，也不会显示对应辅助。
- 如果某个特殊武器助手不想在对局中自动弹出，可以只关闭它，不影响其他助手。

### 5.3 校准区域

见 [第 4 节](#4-首次使用区域校准)。

### 5.4 按键设置

- 找到要修改的功能 → 点击 **录制** → 按下新快捷键 → 点击 **保存快捷键**。
- 点击 **恢复默认** 可重置所有可配置快捷键。
- 如果游戏内修改了打开装备栏按键，必须同步修改这里的「打开装备栏」，否则按下装备栏键后不会触发武器和配件识别。
- 支持录制 `Enter`、方向键、功能键、字母数字键等常用按键；录制后一定要点击保存。

## 6. 压枪参数校准

如果默认压枪效果不好，可以自行校准。进入「启动助手」→ **调试压枪参数**。

压枪的逻辑如下：
+ 首先，突击步枪（包含 MK47、M16A4 可自动连点实行全自动）、冲锋枪、轻机枪这些全自动枪械可持续压枪；射手步枪为点击压枪；狙击步枪无压枪，但有呼吸晃动抑制功能；霰弹枪可使用闪身喷辅助。
+ 全自动枪械与射手步枪需要校准压枪力度
+ 压枪力度这样计算：在每个时间点，武器的压枪力度=裸枪力度*各个配件的补偿参数倍率。
+ 全自动枪械的压枪力度可以随时间进行调节（每个配件也如此）
+ 对压枪力度有影响的配件包括：瞄准镜、枪口、握把、枪托

1. 在游戏中选择枪械，窗口中选择同样的枪械。
2. 按 `F3` 开启/关闭压枪。
3. 拖动曲线点修改对应时间的压枪力度，或修改配件的力度修正系数。
4. 校准压枪时**确保站立姿势**，不要蹲下或趴下。
5. 修改完后点击保存，参数会自动加载并写入配置文件。
6. 推荐先校准裸枪，再逐个添加配件校准。倍镜用红点/全息即可，不影响。
7. 校准完裸枪后可以选择 编辑->其他配件的倍率

### 6.1 SR 呼吸晃动抑制

- 识别到狙击枪并开启辅助压枪后，打开四倍、六倍或八倍镜时会尝试抑制镜内呼吸晃动。
- 使用前需要正确校准对应倍镜的「镜内边」区域。
- 按下 `W/A/S/D` 移动时会暂时停止抑制，松开移动键后再恢复，避免人物移动时准星抽搐。

### 6.2 SG 闪身喷

- 识别到霰弹枪并开启辅助压枪后，状态栏会显示当前探头方向。
- 先手动按 `Q` 或 `E` 选择左/右探头方向；显示 `???` 时不会触发闪身喷。
- 设置好方向后，按鼠标左键会执行一次闪身喷动作。换到非霰弹枪后，状态栏不再显示探头方向。

## 7. 特殊武器校准及使用

本程序支持的特殊武器包括：迫击炮、火箭筒、投掷物（包括手榴弹、燃烧瓶）、VSS、十字弩、C4。

如果默认特殊武器使用效果不好，可以自行校准。进入「启动助手」→ **调试特殊武器**。

校准流程：
1. 选择需要校准的武器
2. 调试对应武器的参数
3. 保存并应用

> 特殊武器校准后点击保存并应用，此时曲线突然动了一下，不必惊慌，是因为重新计算了坐标轴的范围，曲线数据其实正确保存了。

### 7.1 迫击炮

使用方法：
- 先在地图上给目标打标点，开启瞄准辅助。
- 迫击炮助手显示各颜色标点距离，根据目标颜色选择对应距离调整射击参数。检测到垂直测高时，距离会结合高低差补偿。
- 按 `↑` / `↓` 切换当前使用的标点颜色，按 `F6` 可触发迫击炮自动瞄准。
- 自动瞄准会优先使用小地图测距；如果小地图没有检测到当前颜色标点，则使用大地图测距结果。
- 如果当前颜色标点未找到，或距离小于 121m / 大于 700m，屏幕会显示无法瞄准提示。
- 距离不显示时，检查小地图标点是否可见、色盲模式是否正确、区域校准是否准确。

校准方法：
主要是校准垂直测高模块的补偿参数`a`和`b`，用于测量标点的高度进行距离的补偿（参数值越大，补偿的距离越大）

### 7.2 火箭筒

使用方法：
- 手持火箭筒后自动启用。先给目标打标点。
- 屏幕显示辅助标尺，提示不同距离下的瞄准抬高高度。
- 建议使用**第一人称腰射**，不要开镜，以保证标尺位置符合预期。
- 目标与当前位置高度差过大会很不准。尽量选去水平高度基本一致的目标

校准方法：
根据小地图获得的距离信息与准星抬高高度做了一层映射，调整映射曲线的点位进行校准。

### 7.3 投掷物

使用方法：
- 手持投掷物后自动启用。先给目标打标点。
- 屏幕显示抛物线辅助标尺，按目标距离选择对应抬高线。
- 手持手雷且瞄准辅助开启时，按 `B` 可触发**自动瞬爆**流程：根据当前标点距离控制拉环和投掷时机。使用前确认距离识别准确。
- 距离大于 20m 小于 50m 使用直接投掷；距离大于 50m 小于 70m 使用跳跃投掷（跳跃触发也和自动瞬爆一起实现，无需手动跳跃）。
- 如果当前标点未找到或距离不在可用范围内，屏幕会显示红色提示框。

校准方法：
总共含有四个曲线，分别是直接投掷的拉环等待时间和准星抬高高度，以及跳跃投掷的拉环等待时间和准星抬高高度，调整映射曲线的点位进行校准。

### 7.4 VSS

使用方法：
- 手持 VSS 后自动启用。给目标打标点。
- 屏幕显示对应距离的下坠参考线，根据横线位置抬高准星射击。
- 如果未找到 VSS 准星，会在屏幕下方显示红色提示框。

### 7.5 十字弩

使用方法：
- 手持十字弩后自动启用。给目标打标点。
- 标尺以中心左右对称的横线为主，不使用白色竖线；识别到的准星原始位置会显示黑色叉号。
- 如果未找到十字弩准星，会在屏幕下方显示红色提示框。

### 7.6 C4

使用方法：
- 手持 C4 后自动启用。按住鼠标左键安装 C4，安装过程中显示提示。
- 安装完成后 HUD 显示：爆炸倒计时、建议车速、当前使用标点。
- 按 `↑` / `↓` 切换目标标点颜色。
- 建议车速达到合适范围时，HUD 提示推荐起步；距离目标过近时，提示推荐跳车。

校准方法：
调整推荐起步、推荐跳车时的显示时间，分别对应推荐车速、距离。

## 8. 日常使用流程

1. 启动游戏并进入对局。
2. 启动助手主程序。
3. 在「地图点位」中选择当前地图和色盲模式。
4. 按 `F1` 开启武器检测。
5. 按你的「打开装备栏」按键（默认或自定义）打开装备栏，让助手识别主武器和配件。
6. 关闭装备栏，确认状态 HUD 显示的当前武器正确。
7. 按 `F2` 开启瞄准辅助。
8. 按 `F3` 开启辅助压枪（需要时）。
9. 需要大地图测距时按 `F4`。
10. 需要切换目标标点时按 `↑` / `↓`。

> **提示**：每次换枪、换配件后，按你的「打开装备栏」按键打开装备栏让助手重新识别。如果你在游戏内改了装备栏按键，助手里也必须同步修改。

---

## 9. 大地图测距

1. 在游戏中打开大地图，确认瞄准辅助已开启。
2. 按 `F4` 进入大地图测距模式。
3. 按屏幕提示，用鼠标左键点击你在大地图上的**当前位置**。
4. 助手自动寻找黄、橙、蓝、绿四个标点并计算距离，结果显示在显示层中。
5. 测距过程中按 `M` 或鼠标右键可取消本次测距。

> **注意**：大地图测距依赖「大地图区域」和「1km 长度」的校准。如果距离不准，优先重新校准这两项。如果某个颜色没有打标点，对应颜色不显示。

## 10. 常见问题

### 按快捷键没反应

1. 确认游戏窗口处于前台。
2. 确认助手没有正在录制快捷键。
3. 进入「按键设置」检查快捷键是否被改动，或点击「恢复默认」。
4. 仍无效则重启助手。

### 标点识别不到

1. 检查游戏内是否真的打了对应颜色的标点。
2. 检查助手「色盲选择」是否和游戏内一致。
3. 重新校准小地图 / 大地图 / 垂直测高区域。
4. 避免地图被其他窗口、浮层或游戏 UI 遮挡。

### 武器或配件识别错误

1. 打开装备栏后等待状态栏从"武器识别中…"变为稳定结果。
2. 重新校准武器名称、编号和配件区域。
3. 确保游戏语言为中文，装备栏区域没有被浮窗遮挡。
4. 若武器名称仍无法识别，检查「调试缩放比例」中的缩放校准。
5. 如果游戏内打开装备栏按键不是助手中配置的按键，需要先在「按键设置」中同步修改。

### 大地图测距不准

1. 重新校准「大地图」区域和「1km 长度」。
2. 确认助手选择的地图与游戏内一致。
3. 确认点击的是玩家当前位置，而不是目标点。

### 地图点位不显示

1. 确认当前没有识别到手持武器（识别到武器时会自动关闭）。
2. 按住 `鼠标左键 + 中键` 触发显示。
3. 检查是否选择了正确地图，以及点位类型是否被全部关闭。

### 特殊武器助手不显示

1. 确认「瞄准辅助」已开启（`F2`）。
2. 确认「武器检测」已开启（`F1`）。
3. 确认状态 HUD 显示的当前武器识别正确。
4. 检查「启动助手」中对应特殊武器按钮是否处于开启状态。

### 压枪不生效

1. 确认「武器检测」（`F1`）和「辅助压枪」（`F3`）都处于开启状态。
2. 打开装备栏重新识别配件。
3. 姿势影响不正确时，重新校准「姿势区域」。
4. 换枪后仍按旧武器压枪时，打开装备栏重新识别。
5. 确认游戏内灵敏度为默认值；灵敏度不是默认值时，压枪力度会明显偏差。
6. 手持特殊武器（火箭筒/投掷物/VSS/十字弩/C4）时压枪自动不生效，这是正常现象。

### 特殊武器辅助位置不准

1. 确认游戏内所有灵敏度均为默认值。
2. 确认小地图、大地图、垂直测高、准星区域已经按当前分辨率重新校准。
3. 确认助手色盲模式和游戏色盲模式一致。
4. 火箭筒、投掷物、VSS、十字弩都依赖当前标点距离，标点识别不准时辅助位置也会不准。
