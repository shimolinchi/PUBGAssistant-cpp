# PUBGAssistant-cpp 使用说明

主程序窗口包含四个选项卡：地图点位、启动助手、调试校准、使用说明。完整说明文档由程序同目录下的 `README.pdf` 提供，点击“查看完整使用说明文档”会直接打开这个 PDF。

## 首次使用

1. 选择游戏中设置的色盲模式。
2. 校准区域框、校准缩放比例。
3. 修改快捷键及游戏中设定的按键必须一致。
4. 开火不能用鼠标左键，需要在游戏中调成其他按键，并在程序中同步修改“开火按键”。
5. 游戏中调整所有灵敏度为默认灵敏度，否则需要全部重新调整压枪参数。

## 游戏内建议设置

- 游戏语言建议使用中文，武器名和装备栏识别依赖中文模板。
- 显示模式建议使用全屏或无边框全屏，避免截图区域漂移。
- 关闭小地图动态缩放，保证测距比例稳定。
- 如果修改了打开装备栏、开火、打开小地图等游戏按键，必须进入“修改快捷键及游戏按键”同步修改。
- 更换分辨率、游戏 UI 比例、显示器缩放后，需要重新校准区域框和缩放比例。

| --- | --- | --- |
| 武器检测开关 | F1 | 控制装备栏、手持武器和姿势识别。 |
| 测距瞄准显示开关 | F2 | 特殊武器和测距显示层总开关。 |
| 辅助压枪开关 | F3 | 控制全自动压枪、连狙点压、狙呼吸稳定、自动闪身喷。 |
| 大地图测距 | F4 | 打开大地图测距流程。 |
| 迫击炮自动瞄准（长按） | 鼠标右键 | 上炮后长按触发自动调高和方向修正。 |
| 窗口显示开关 | Home | 显示或隐藏主窗口。 |
| 打开装备栏 | 按配置，常用 Tab/Enter | 按下后触发装备栏确认和武器配件识别。 |
| 打开小地图 | N | 程序自动打开/关闭游戏小地图时使用，也可手动按下切换特殊武器显示。 |

| 标点前后切换 | 上 / 下 | 切换当前选中的标点颜色。 |
| 投掷物 | B | 手雷自动瞬爆相关按键。 |

## 修改快捷键及游戏按键

在“调试校准”中点击“修改快捷键及游戏按键”打开快捷键窗口。

- 点击某一行的“录制”，按下新按键，再点击“保存快捷键”。
- “打开装备栏”“打开小地图”“打开大地图”“开火按键”“腰射瞄准”“投掷物”建议使用单键，并与游戏内设置保持一致。
- “打开小地图”默认是 `N`。特殊武器识别到后，程序会使用这个按键打开游戏小地图；切回普通武器或手动关闭时也会使用它。
- 如果使用 PowerToys 修改了 F1~F8 等功能键，录制时仍应按实际硬件按键，让程序按硬件状态识别。

## 调试校准

调试校准页包含：

- 调试压枪参数
- 调试特殊武器
- 显示所有区域框
- 校准区域框窗口
- 调试区域缩放比例
- 修改快捷键及游戏按键
- 自定义显示界面

首次使用或更换分辨率后，建议先完成区域框校准，再进行缩放比例校准。小地图、大地图、顶部方向、迫击炮上炮区域、装备栏区域和姿势区域都需要尽量框准。

## 特殊武器助手

启动助手页中的特殊武器按钮是允许/禁止开关。默认开启；关闭某个按钮后，即使识别到对应武器，也不会触发该助手。

特殊武器助手依赖标点颜色和测距结果。识别异常时，优先检查：

- 游戏色盲模式和程序色盲模式是否一致。
- 小地图或大地图区域是否校准准确。
- 当前颜色标点是否真的存在且没有被遮挡。
- “打开小地图”按键是否和游戏内按键一致。

## 迫击炮

- 迫击炮助手不是显示层开启就直接显示，而是需要识别到上炮状态后才显示。
- 按 F 上炮后，程序会检测迫击炮上炮区域；识别成功后切换为迫击炮状态并显示测距。
- 下炮或识别到其他武器后，会关闭迫击炮显示。
- 迫击炮自动瞄准默认长按鼠标右键触发，只有上炮后才会生效。
- 如果按 ESC 打开游戏系统菜单，程序会暂时保持迫击炮状态，避免在菜单界面误按小地图键导致时序错乱。

## 压枪与呼吸晃动稳定

- 游戏内所有灵敏度必须保持默认，否则压枪和特殊武器抬高位置都会偏。
- 开火按键必须与游戏内一致，且不能是鼠标左键。
- SR 呼吸晃动稳定需要正确校准对应倍镜的镜内上边缘区域。
- 按下 WASD 移动时会暂时屏蔽呼吸晃动稳定，松开后恢复。
- SG 自动闪身喷只在识别到霰弹枪、压枪开启、并且已通过 Q/E 记录探头方向时触发。

## 完整 PDF 文档

程序不会再内置重新写出的完整说明，而是读取 exe 同目录下的 `README.pdf`。

- Debug 运行时：将 `README.pdf` 放到 `.build/debug/README.pdf`，或放到项目根目录后重新编译，让构建自动复制。
- Release 打包时：如果项目根目录存在 `README.pdf`，会自动复制到 `.build/package/release/README.pdf`。
- 点击“查看完整使用说明文档”时，如果同目录没有 `README.pdf`，程序会提示缺少文件。
